# mcworld转json可编辑的命令方块文本文件
# 基岩版命令方块提取/回写工具

双击即可打开使用，**无需第三方库**（纯 Python 标准库实现），也无需安装 Java、Node 等任何环境。
该工具可以将你的存档转化成可编辑的json文件，你可以在里面修改你的命令方块文本，以及它的悬浮字！修改后的json文件可以回转到mcworld，可以导入在你的Minecraft中！

## 快速开始

### 前提：需要 Python 3

本工具需要电脑上安装了 Python 3.9 或更高版本。

- **Windows**：去 https://www.python.org/downloads/ 下载安装包，安装时**务必勾选
  "Add python.exe to PATH"**，并保持默认勾选 "tcl/tk and IDLE"（图形界面依赖它，
  官方安装包默认就是勾选的，一般无需手动处理）。
- **macOS**：系统自带 Python3 通常没有图形库，建议用 Homebrew 安装：
  `brew install python-tk`，或直接去官网下载安装包（自带 tkinter）。
- **Linux**：大部分发行版需要额外装一下：
  `sudo apt install python3-tk` (Ubuntu/Debian) 或 `sudo dnf install python3-tkinter` (Fedora)。

### 启动方式

- **Windows**：双击 `启动程序.bat`
- **macOS**：终端里执行 `chmod +x 启动程序.command` 一次后，双击 `启动程序.command`
  （或直接在终端里 `./启动程序.command`）
- **Linux / 通用方式**：命令行运行 `python3 main.py`

程序启动时会**自动检查依赖**：
- 本工具核心的存档解析逻辑是纯 Python 标准库实现，**不需要安装任何 pip 第三方包**，
  开箱即用。
- 如果检测到系统缺少 tkinter 图形库，会给出安装提示。

## 使用流程

### 提取命令方块

1. 在"提取命令方块"标签页，点击"浏览…"选择你的 `.mcworld` 存档文件。
2. 指定输出 JSON 路径（默认会自动填充为 `<存档名>_commands.json`）。
3. 点击"开始提取"。提取完成后，命令方块列表会显示在下方表格中，可以直接编辑。

### 编辑命令文本

- 在列表中选中一行，在下方的输入框中修改命令文本，然后点击"应用修改"。
- 修改完后点击对应按钮保存到文件。

### 回写存档

1. 切换到"回写存档"标签页。
2. 选择原始存档（作为基础副本）。
3. 选择修改后的 JSON 文件。
4. 指定输出新存档路径。
5. 点击"开始回写"。完成后会得到一个**全新的** `.mcworld` 文件
   （不会覆盖你的原始存档，可以放心尝试）当然你可以修改它的名称。

## 安全性说明

- 回写生成的是全新文件，不会覆盖或损坏你的原始存档。
- 技术上采用"追加新 WAL 日志文件"的方式写入修改，不触碰存档原有的核心数据文件，
  这是 LevelDB（存档使用的底层数据库格式）官方支持的标准写入机制，安全可靠。
- 不放心的话，建议先在存档的备份副本上试用。

## 支持的 Minecraft 版本

- ✅ 基岩版 1.16 ~ 1.21+（含 Caves & Cliffs 格式）
- ✅ NetEase 中国版基岩版
- ✅ 主世界 / 下界 / 末地 全维度
- ⚠️ 如果遇到问题，欢迎提交issue！

## 文件清单

| 文件 | 说明 |
|---|---|
| `main.py` | 主程序入口，双击/命令行运行这个 |
| `gui_app.py` | 图形界面代码 |
| `启动程序.bat` | Windows 双击启动脚本 |
| `bnbt.py` | 基岩版 NBT 编解码器（核心解析逻辑） |
| `mcleveldb.py` | 基岩版存档 LevelDB 读取器 |
| `mcleveldb_write.py` | 安全写入模块 |
| `cb_extract.py` | 提取命令方块的业务逻辑 |
| `cb_repack.py` | 回写 JSON 的业务逻辑 |
